Le dossier Scripts contient l\textquotesingle{}ensemble des fichiers de code sauf \textquotesingle{}\mbox{\hyperlink{GamePlay_8cs}{Game\+Play.\+cs}}\textquotesingle{}

Le fichier \textquotesingle{}\mbox{\hyperlink{GamePlay_8cs}{Game\+Play.\+cs}}\textquotesingle{} contient la logique de jeu.

Dans le dossier Scripts/\+Database, une version de la base de donnée est contenue.

Le dossier Game\+Cinematics contient l\textquotesingle{}ensemble des fichiers pour la cinématics du jeu.

Le dossier Pictures\+Patients contient les images des patients du jeu.

Le dossier Ressources contient les autres fichiers de ressource nécessaire au jeu.

Le dossier Config contient un fichier \textquotesingle{}gameconfig.\+cfg\textquotesingle{}, le fichier de config pour le jeu. Mais aussi le fichier \textquotesingle{}exemple.\+cfg\textquotesingle{}, un fichier d\textquotesingle{}exemple contenant l\textquotesingle{}ensemble des pâramètre possible.

Le dossier doc contient la documentation du code format Web.

Le dossier diagramme contient le diagramme uml et le mcd de la base de donnée.
