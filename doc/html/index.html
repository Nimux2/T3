Explication de la structure des fichiers

Le dossier Scripts contient l'ensemble des fichiers de code sauf 'GamePlay.cs'

Le fichier 'GamePlay.cs' contient la logique de jeu.

Dans le dossier Scripts/Database, une version de la base de donnée est contenue.

Le dossier GameCinematics contient l'ensemble des fichiers pour la cinématics du jeu.

Le dossier PicturesPatients contient les images des patients du jeu.

Le dossier Ressources contient les autres fichiers de ressource nécessaire au jeu.

Le dossier Config contient un fichier 'gameconfig.cfg', le fichier de config pour le jeu. Mais aussi le fichier 'exemple.cfg', un fichier d'exemple contenant l'ensemble des pâramètre possible.

Le dossier doc contient la documentation du code format Web.

Le dossier diagramme contient le diagramme uml et le mcd de la base de donnée.
